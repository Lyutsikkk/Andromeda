/datum/component/egg_layerУправляет количеством яиц, которые можно отложить, чем можно кормить моба для увеличения кладки, и что именно откладывается. Поскольку основное взаимодействие с компонентом - это attackby, мы можем сделать это процедурой уровня атома. egg_layer будет явно выдавать ошибки при отсутствии аргументов, чтобы обеспечить ясность.
