/tg/ Station 13 - Modules - Types — Var Details - Proc Details

security_level
/datum/security_level

Уровни безопасности

Используются подсистемой уровней безопасности. Каждый из них представляет уровень безопасности, который может установить игрок.

Базовый тип является абстрактным

Vars

announcement_colorЦвет разделителя нашего объявления.
disables_mailесли TRUE, блокирует отправку почтовых отправлений во время этого уровня безопасности
elevating_to_announcementНаше объявление при повышении до этого уровня
elevating_to_configuration_keyКлюч конфигурации для текста повышения, если установлен, переопределит стандартное объявление повышения.
fire_alarm_light_colorЦвет света пожарной тревоги, устанавливаемый при изменении на этот уровень безопасности
looping_soundЗацикленный звук, который будет воспроизводиться пока установлен уровень безопасности
looping_sound_intervalИнтервал зацикленного звука
lowering_to_announcementНаше объявление при понижении до этого уровня
lowering_to_configuration_keyКлюч конфигурации для текста понижения, если установлен, переопределит стандартное объявление понижения.
nameНазвание этого уровня безопасности.
name_shortformТрёхбуквенное сокращение уровня безопасности.
number_levelЧисловой уровень этого уровня безопасности, см. defines для дополнительной информации.
set_delayЗадержка перед установкой этого уровня безопасности
shuttle_call_time_modМодификатор времени вызова шаттла для этого уровня безопасности
soundЗвук, который будет воспроизведён при установке этого уровня безопасности
status_display_icon_stateСостояние иконки, которое будет отображаться на дисплеях во время этого уровня безопасности

Procs

post_set_security_levelВызывается после установки уровня безопасности, прямо перед отправкой COMSIG_SECURITY_LEVEL_CHANGED
pre_set_security_levelВызывается перед установкой или планированием установки уровня безопасности
