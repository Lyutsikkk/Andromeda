Vars

always_celebrateдля вечного рождества или тестирования. Принудительно отмечает праздник.
begin_dayКакой день begin_month является началом праздника?
begin_monthВ каком месяце начинается праздник?
current_yearХранимая переменная для лучшего расчёта дат определённых праздников, например пасхи.
end_dayКакой день end_month является концом праздника? Значение по умолчанию 0 означает, что праздник длится один день.
end_monthВ каком месяце заканчивается праздник?
holiday_colorsЦветовая схема этого праздника
holiday_hatЕсли определено, дроны/ассистенты без стандартной шляпы будут спавниться с этим предметом в слоте головного убора.
holiday_mailСписок предметов, добавляемых в пул почты. Может быть взвешенным списком или обычным списком. Оставьте пустым для отсутствия.
holiday_patternСтандартный узор праздника, если запрашиваемый узор равен null.
nameНазвание самого праздника. Видно игрокам.
no_mail_holidayКогда этот праздник активен, предотвращает ли он доставку почты в карго? Переопределяет var/list/holiday_mail. Старайтесь не использовать для длительных праздников.
timezonesЧасовые пояса, в которых отмечается этот праздник (по умолчанию три часовых пояса охватывают 50-часовое окно для всех часовых поясов)
year_offsetНа сколько лет смещаются расчёты для begin_day и end_day. Используется для праздников вроде пасхи.
