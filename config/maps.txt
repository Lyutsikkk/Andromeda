# Этот файл содержит список карт для ротации
# Строки, начинающиеся с #, игнорируются
# Формат:
# map [название карты] (имя .json файла из папки _maps без расширения)
#	minplayers [число] (0 или меньше отключает ограничение)
#	maxplayers [число] (0 или меньше отключает ограничение)
#	default (последняя карта с этой пометкой получит голоса игроков, не выбравших конкретную карту)
#	voteweight [число] (вес голоса, по умолчанию 1. 0.5 = полголоса, 2 = двойной вес и т.д. 0 отключает карту из голосования, но оставляет доступной для ручного выбора)
#	disabled (отключает карту)
#	votable (участвует в голосовании)
#	feedbacklink (ссылка для отзывов о карте)
#	webmap_url (ссылка на веб-карту)
# endmap
