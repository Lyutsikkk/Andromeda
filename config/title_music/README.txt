```txt
Папка /sounds содержит звуковые файлы, используемые в качестве заглавной музыки игры. Поддерживаются форматы OGG и WAV.

Рекомендации:
- Избегайте использования чрезмерно больших звуковых файлов (могут вызывать лаги у клиентов)
- Вы можете добавить любое количество музыкальных тем для загрузочного экрана
- Если добавлено несколько файлов, игра будет выбирать их случайным образом (см. правила именования)

---

Правила именования:

Каждый звуковой файл должен иметь уникальное имя. Хотя технически возможно использовать одинаковые имена для файлов разных типов, это не рекомендуется.
Избегайте использования знаков "+" и "." в именах, так как они зарезервированы системой.

Обычные музыкальные темы:
Используются в стандартной ротации. Любое имя без символа "+" считается обычной темой.
Пример: "clown.ogg"

Музыка для конкретных карт:
Связывается с определённой игровой картой. Формат имени:
"(название карты)+(название звука)"

Важно:
- Название карты должно точно соответствовать MAP_NAME из соответствующего .DM файла в папке /_maps
- Можно увидеть название карты в игровом меню статуса
- Не используйте пробелы между частями имени

Пример: "Omegastation+splash.ogg"
Допускается:
- Несколько музыкальных тем для одной карты
- Редкие музыкальные темы для карт

Редкие музыкальные темы:
Специальные темы с 1% шансом проигрыша в каждом раунде.
Формат имени: "rare+(название темы)" (без пробелов)
Пример: "rare+explosion.wav"
```
