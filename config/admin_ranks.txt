# Ниже приведены допустимые ключевые слова прав:
#ADMIN = Общие админ-инструменты — доступ к панели админа, ахелпам, базовым командам.
#FUN = Ивент-команды — доступ к "веселым" секретам и возможностям (может быть опасно!)
#BAN = Баны — возможность банить и разбанивать игроков.
#STEALTH = Скрытый режим — появляться в чате под фейковым именем (видно только админам)
#POSSESS = Обладание объектами — возможность вселяться в предметы/мобов.
#POLL = Создание опросов — запуск голосований на сервере (требует БД).
#BUILD = Режим строительства — доступ к Build Mode.
#SERVER = Управление сервером — рестарт, смена режима игры, принудительное начало/конец раунда
#DEBUG = Дебаг-инструменты — доступ к отладке кода, просмотру логов.
#VAREDIT = Редактор переменных — изменение любых переменных объектов (опасно!)
#PERMISSIONS = Назначение прав — повышение/понижение других админов.
#SOUND = Управление звуками — загрузка и воспроизведение звуковых файлов.
#SPAWN = Спавн объектов — создание предметов, мобов (кроме опасных, например, черных дыр)
#AUTOADMIN = Автоподключение прав — автоматическая выдача прав при входе (по умолчанию вкл)
#DBRANKS = Сохранение прав в БД — запись изменений прав в базу данных (требует SQL)
#EVERYTHING = Все права — включает абсолютно все доступные флаги
#@ = Наследование прав — копирует все флаги из предыдущего ранга в текущей категории.
